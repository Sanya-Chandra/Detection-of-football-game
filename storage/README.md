# 📂 Media Storage Workspace

This is your dedicated workspace for storing and managing sports media.

## 📁 Directory Structure

```text
storage/
├── images/
│   ├── input/    <-- Place raw sports images here (.jpg, .png, .webp)
│   └── output/   <-- Processed/annotated images are saved here
│
└── videos/
    ├── input/    <-- Place raw sports videos here (.mp4, .avi, .mov)
    └── output/   <-- Processed/annotated videos are saved here
```
